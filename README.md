**SCISSOR**
This is a url shortner app created with branded custom slugs, QR code generation, and click analytics.
**Fullstack** description below
Frontend - React + Typecript
Backend - Convex
Autentication -Clerk + Convex Integration
QR code- qrcode.react
Redirects - Convex HTTP Actions
Analytics - Convex mutations + schedule functions
Charts - Recharts
Styling - Tailwind CSS
Deployment - Vercel + Convex cloud
